# 通用验收标准

> 模块特有的验收标准在各 `prompts/P-XX.md` 中，本文件只放跨模块通用标准。

## 文档结构
- [ ] 所有输出文档使用 Markdown，含清晰的二级/三级标题
- [ ] 文件命名遵循 0X-description.md 格式
- [ ] 文档末尾含"最后更新"时间戳

## 一致性要求
- [ ] 所有文档中的项目名称、上线日期保持一致
- [ ] 输出文件中的外部依赖名称与 CLAUDE.md 对齐
- [ ] 竞品名称在所有文档中保持一致
- [ ] 待人工确认项总数 ≤ 10 条

## 提交规范
- [ ] 每个文件生成后立即保存到 docs/app-launch/ 目录
- [ ] 每次修改在文档末尾追加变更日志
